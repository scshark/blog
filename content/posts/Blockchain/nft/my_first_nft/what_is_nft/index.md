# <center > <font color="#377dff"> 什么是 NFT？</font> </center> {#what_is_nft}
</br>

## 同质化代币（FT）和非同质化代币（NFT）区别？{#ft_and_nft}
> + **同质化代币（FT，全称 Fungible Token）是可以被任意交换的代币，就像钱一样。**
> + **非同质化代币（NFT，全称 Non-fungible Token）则是独一无二、完全不同，难以用来平等交换的代币。**

{{< image src="https://blog.cdn.scshark.com/scshark/2022-09-20/16636438653816.jpg" caption="NFT" >}}


## NFT 特性和价值 {#nft_cv}

**<center> 那么我们能从基于区块链技术的 NFT 得到什么呢？ </center>**

> * **独一无二，不可分割**
>  
>   每一个 NFT 都不可分割，独一无二，无法等价交换。
> 
> 
> * **不可篡改，公开透明**
>  
>   NFT 基于区块链技术，任何人都可以查看且不可篡改。
> 
> * **万物皆可 NFT**
>  
>   NFT 可以在虚拟世界代表任意非钱币的实体。
> 
> * **完全控制权**
>  
>   NFT 所有者将对其拥有永久、完全的控制权，不可被剥夺。（大部分数字藏品并不算真正的 NFT，因为不在公链上，可以被平台控制）
> 
> * **确权成本低**
>  
>   验证某人是否拥有某个 NFT 的成本非常低，可以快速确认版权归属。
> 
> * **无限想象空间**
> 
>   NFT 的具体功能和玩法由参与者定义，具有无限想象空间。
>


### NFT 在未来世界的应用 {#nft_feture}

{{< admonition  type="summary" title="The world of the future">}}

* **游戏**
 
    > **现状**
    > 
    > 游戏里的虚拟资产仅限于当前游戏，无法在游戏之间流通，且受限于游戏平台规则，所以玩家很难进行自由买卖。另外游戏账号也并非被你拥有，一旦被封号你将失去所有游戏资产。
    >    
    > **未来**
    >
    > 结合区块链和 NFT 技术，开发者可以设计出具有唯一性的游戏虚拟资产，且完全归属于个人，例如账号、武器、工具、房屋、宠物等，玩家可以自由的进行游戏资产的交易和流通。

* **版权确认**
 
    > **现状**
    > 
    > 被盗版之后，很难确权，需要准备大量文件和记录。导致小规模盗版事件频发也无可奈何。
    >    
    > **未来**
    >
    > NFT 并不能解决盗版问题，但是相比传统的盗版艰难的反侵权诉讼而言，仅需要几秒即可确定你是否拥有版权。也非常容易确认谁最先创造了这个物品。未来正规商业公司在采用素材时，非常容易确认谁是所有者，提升商业合作效率降低风险和盗版事情的发生。


* **买卖房产**
 
    > **现状**
    > 
    > 买卖房产需要聘请律师，通过政府信用背书的房产证等资料来确认房产所有者，甚至需要由政府档案处管理资料。这要付出时间、金钱、和一个可信任的第三方才能完成交易。
    >    
    > **未来**
    >
    > 如果未来 NFT 得到了法律认可，代表了你的房产，那么只需要几分钟转移 NFT 即可完成房产所有权交割。

* **艺术品收藏**
 
    > **现状**
    > 
    > 艺术品通常为实物，需要妥善保管避免被盗和损坏。展示时需要较大成本来运输和做安保工作。
    >    
    > **未来**
    >
    > 艺术品为虚拟物品，基于 NFT 技术确保艺术品的安全性。在钱包安全的情况下，无需担心艺术品被盗火者损坏。在元宇宙中可以任意展示你的藏品或者举办艺术展。想要提前体验一下元宇宙 NFT 艺术展？可以访问  {{< link "https://oncyber.io/6529om" "https://oncyber.io/6529om">}}

* **交换物品**
 
    > **现状**
    > 
    > 交易和交换的信任成本很高，往往需要引入第三方可信平台作为担保执行。需要一定的时间和金钱成本。
    >    
    > **未来**
    >
    > 通过区块链和 NFT 技术，可以很轻松的查看到过去的交易记录来确定物品的来源和真实性。通过智能合约等技术，可以直接实现交易和交换。减少诈骗、交易纠纷和信任成本。
    
* **通行证**
 
    > **现状**
    > 
    > 不同的系统的门票、会员卡、员工卡、通行证不互通无法相互使用，每个系统需要自己准备系统门票和会员卡等。
    >    
    > **未来**
    >
    > 所有系统门票、会员卡、员工卡、通行证均通过 NFT 技术实现。每个人的身份与其钱包绑定，不同的系统可以非常快速准确的识别你的身份，并为你提供定制化的服务。

### NFT 的现状 {#nft_stauts}

**NFT 还处于非常早期的阶段**

NFT 在 2021 年开始爆发，目前大概市场规模在 180 亿美元。市场预期在 2026 年到达 820 亿美元的规模。 

诸多知名品牌和明星，例如：周杰伦、余文乐、阿迪达斯、古驰等等，也在开始试水发行或者购买 NFT。

{{< image src="https://blog.cdn.scshark.com/scshark/2022-09-20/16636572075471.jpg" caption="Data from The Block Crypto Data." >}}

**目前大部分 NFT 是头像类型的项目**

由于 NFT 所代表的是虚拟网络的物品，比较抽象。因此目前比较常见的火爆的 NFT 项目是一些艺术作品，包括头像、画作、3D 艺术品、照片等等。

也开始逐步出现更多 NFT 的玩法和项目，比如社区通行证、游戏道具、门票等。

{{< image src="https://blog.cdn.scshark.com/scshark/2022-09-20/16636574577550.jpg" caption="Avatar Type NFT " >}}

**<center > 整体而言，NFT 市场处于非常早期的阶段，作品质量参差不齐鱼龙混杂。所以在参与购买 NFT 的时候，需要 DYOR。</center>**

### 什么是 DYOR？如何做 DYOR？{#what_is_DYOR}

**在 Web3 你会随处可见 DYOR。它的全称是 Do your own research，意思是你要自己做研究来决定是否购买，是一个免责声明。那么如何做 DYOR？**


> :one: **在决定购买 NFT 之前，您需要考虑清楚你为什么购买？**
>
>>  :white_check_mark: 因为这个艺术品跟你产生了共鸣？你非常想要收藏拥有它？
>>  
>>   :white_check_mark: 因为你看好这个 NFT 背后的社区和文化，想要加入他们？
>>  
>>   :white_check_mark: 因为你研究分析认为这个项目方非常有实力，未来会有升值空间？
>>  
>>   :x: ~~因为看到群里和网上有很多人宣称这个肯定赚钱 FOMO 了，不管怎样自己一定要买!~~


> :two: **你需要尽可能多的了解这个 NFT 项目的一切**
>  
>> 你需要尽可能多的了解这个项目的一切信息，包括但不限于：
>> * 作品预览效果（sneak-peaks）
>> * 项目组成员和背景
>> * 项目未来规划和落实的可能性
>> * 你朋友的看法


> :three: **评估当前价格对于你是否合理**
>  
>> NFT 由于其限量且独一无二的特性，非常容易被炒作。为了避免不必要的损失，建议你对于价格进行预估。
>> * 你认为这个 NFT 项目具备什么价值？纪念价值？艺术价值？工具价值？社区价值？
>> * 你愿意为此付出多少钱？

{{< admonition type="tip" title="行业黑话" >}}

* FOMO，全称 Fear of missing out，意思是害怕失去机会。在这种情绪下，往往会高价接盘，容易被套牢。
  
* FUD，全称 Fear, uncertainty, and doubt，意思是害怕猜疑不确定。在这种情绪下，往往会低价出货，容易卖飞了。
